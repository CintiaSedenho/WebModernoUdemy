Arrow Functions
A arrow function (ou função seta) é uma forma mais concisa de declarar funções em JavaScript. Introduzida no ES6 (2015), ela é especialmente útil para funções curtas e funções de callback.

✅ Sintaxe básica
const saudacao = (nome) => {
  return `Olá, ${nome}!`;
};

Se a função tiver apenas um parâmetro e uma linha de retorno, você pode simplificar ainda mais:

const saudacao = nome => `Olá, ${nome}!`;

⚠️ Limitações das arrow functions
Não tem this, arguments, super nem new.target próprios.
Não pode ser usada como constructor (você não pode usar new com elas).
Menos adequada para métodos de objeto que precisam de this dinâmico.

🧪 Exemplos práticos
Map com arrow:
const numeros = [1, 2, 3];
const dobrados = numeros.map(n => n * 2);
console.log(dobrados); // [2, 4, 6]

Filtro com arrow:
const nomes = ['Ana', 'João', 'Clara'];
const curtos = nomes.filter(nome => nome.length <= 4);
console.log(curtos); // ['Ana', 'João']
